# Description

InboxShield is a privacy-first multi-agent system that safely triages untrusted
customer-support messages. It redacts sensitive information, detects prompt
injection, assigns urgency, retrieves policy through MCP-compatible tools,
drafts a safe response, and keeps consequential actions behind human approval.

Built for Kaggle's AI Agents: Intensive Vibe Coding Capstone Project 2026,
Agents for Business track.
